***

一个非常小型的`IA-32`架构操作系统。

本人的第二版的操作系统，对第一版进行了大量的修改，对某些地方进行重新设计，添加了更多驱动的支持，兼容APIC与PIC等。

会抽时间不断的完善该操作系统，并不断地增加更多的功能。

***

## 目录层次

```
.
├── conf                //  配置文件
├── docs                //  参考文档
└── src                 //  源码目录
    ├── arch            //  架构代码
    │   ├── i386
    │   └── x86_64
    ├── boot            //  引导程序
    ├── kernel          //  内核代码
    └── libk            //  内核库
```
